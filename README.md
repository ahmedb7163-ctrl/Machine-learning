I'm excited to share my latest project in HR Analytics, where I built a Machine Learning model that predicts the likelihood of employee attrition with high accuracy! 

Using Python, I analyzed and preprocessed employee records from the "employee_data.csv" dataset.

Key Insights:

Satisfaction is Key : Employees who left had an average satisfaction level of only 44%, compared to 67% for those who stayed.

Handling Class Imbalance: The target variable was highly imbalanced (82.7% stayed). I successfully addressed this by applying SMOTE to balance the training data.

Modeling & Results:

After cleaning and scaling the features, I trained a Decision Tree Classifier and achieved outstanding results:

Test Accuracy: 94.4%! 

F1-Score: 0.94 for both classes, proving the model is highly robust.
